# -STRIVE- Silli Secondary Picker
A mathematical secondary picker for GUILTY GEAR -STRIVE-, now with zero vibe coding!

## Hosting a fork of this bot
1. Create a `config.json` file in the root directory
2. In it, include your bot token as well as your client ID like this:
```
{
    "token": "your-token-here",
    "clientId": "your-clientid-here",
}
```
3. Run `fetch_data.js` in a Node terminal to fetch the puddle-farm matchup and character data required to make the bot work
4. Run `index.js` to start the bot
